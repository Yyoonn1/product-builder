# Movie Finder

장르별 영화 찾기 웹사이트입니다.

## 기능
- 장르별 영화 필터링 (액션, 로맨스, SF 등)
- 반응형 영화 카드 그리드 UI
- 다크 모드 테마

## 설치 및 실행
이 프로젝트는 정적 웹사이트입니다. `public/index.html` 파일을 브라우저에서 열어 바로 확인하거나, 로컬 서버를 통해 실행할 수 있습니다.

### Firebase 배포
Firebase CLI가 설치되어 있다면 아래 명령어로 배포할 수 있습니다.

```bash
firebase login
firebase init hosting
# (기존 public 폴더를 사용한다고 선택하세요)
firebase deploy
```
